== Description ==

Poster Twitter WP publish twitter automatic, simple how to program your tweet and publish it every so often
Alternate tweets in a random way to prevent spam, so simple but very useful


== Installation ==


1. Download the plugin
2. Enter the administrator of your WordPress.
3. Enter Plugins / Add-New / Upload-Plugin.
4. Find the plugin downloaded on your computer and upload it like any other file.
5. After installing the .zip you can see it in the list of installed plugins, you can activate or deactivate it.
6. Create an app in https://apps.twitter.com and insert the url callback given in the plugin field
7. You must allow the use of the plugin to users with the role of subscriber

== Frequently Asked Questions ==

= how often the default tweets are published ? =

Every 30 minutes

= What else should I keep in mind? =

**About scheduled tasks, you must create a cron job on a server so that the publications are given**
**The minimum time for the cron job should be 5 minutes**
